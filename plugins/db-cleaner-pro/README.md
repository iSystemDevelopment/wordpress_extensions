# DB Cleaner Pro

Safe WordPress database maintenance — transients, orphans, spam/trash. Does not touch SEO URLs or taxonomy.

- **License:** MIT (repo root)
- **Cron:** weekly (`weekly_database_cleanup`)
- **Log:** `wp-content/db-cleaner-pro.log`
- **Admin:** Tools → DB Cleaner

Install: copy folder to `wp-content/plugins/db-cleaner-pro/` and activate.
